This class controls the command line interface for the user. When the program runs with no arguments a list
of available files should be printed to the screen. When the program runs with just a number as an argument
the contents of the corresponding file should be displayed on the screen. When no second parameter is
given the program uses the default key. If a second parameter is specified this indicates which cipher to
use to decode the file.

Userinterface Pseudocode:


if arguments == 0:
    list file names available
    program exits

if arguments == number:
    output the contents of the specified file using the default cipher
    exit program

if arguments == 2:
    output the contents of the specified file using the specified cipher
    exit program

if arguments >= 3:
    output error "Too many positional arguments"
