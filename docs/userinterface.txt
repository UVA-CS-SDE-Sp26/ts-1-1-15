This class controls the command line interface for the user. When the program runs with no arguments a list
of available files should be printed to the screen. When the program runs with just a number as an argument
the deciphered contents of the corresponding file should be displayed on the screen. When no second parameter is
given the program uses the default key. If a second parameter is specified this indicates which key to
use to decipher the file.

Userinterface
-------------------------------
+String determineOutput(String[])

Userinterface Pseudocode:
-------------------------------

determineOutput(array args) -> String:

    if args.length == 0:
        list all available files
        return

    else if args.length == 1:
        try:
            decipher with default key
            return
        catch:
            return "Error: First argument must be the number of the file you wish to decipher"

    else if args.length == 2:
        try:
            decipher with specified key
        catch:
            print "Error: First argument must be the number of the file you wish to decipher"

    else:
        throw illegal argument exception "Too many arguments called with program"
